# 更新日志

## [1.0.0] - 2025-11-04

### 首次发布 🎉

#### 核心功能
- ✅ 基于斐波那契回调的量化交易策略
- ✅ 金字塔式分批建仓系统
- ✅ baostock数据集成
- ✅ XTTrader交易接口支持
- ✅ Flask RESTful API
- ✅ 现代化Web界面

#### 策略特性
- 自动查找最近60天涨停记录
- 计算斐波那契0.382/0.5/0.618/0.7/0.786回调位
- 第一阶段: 0.5-0.618回调，5笔订单，70%仓位
- 第二阶段: 0.618-0.7回调，3笔订单，30%仓位
- 智能股数计算（向下取整到100的倍数）

#### API端点
- `POST /api/analyze` - 分析股票并生成订单
- `POST /api/execute` - 执行订单（模拟/真实）
- `GET /api/history` - 查询历史订单
- `GET /api/statistics` - 获取统计信息
- `GET /api/order/:id` - 获取订单详情
- `DELETE /api/order/:id` - 删除订单记录
- `GET /api/health` - 健康检查

#### 前端功能
- 📊 直观的数据展示
- 💼 订单列表和详情
- 📈 实时分析结果
- 🎨 响应式设计
- 📱 移动端适配

#### 技术栈
- **后端**: Flask 3.0, Python 3.8+
- **数据**: baostock 0.8.9, pandas, numpy
- **前端**: HTML5, CSS3, JavaScript
- **交易**: XTTrader integration

#### 配置选项
- 可配置的查询天数
- 可配置的涨停阈值
- 可配置的仓位分配比例
- 可配置的回调区间
- 模拟模式/真实交易模式切换

#### 工具和脚本
- `start.sh` - 一键启动脚本
- `test_strategy.py` - 策略测试脚本
- `Dockerfile` - Docker容器化支持
- `docker-compose.yml` - Docker Compose配置

## 已知问题

### 数据相关
- baostock数据有约15分钟延迟
- 部分ST股票涨停阈值识别可能不准确

### 性能相关
- 首次查询数据时可能较慢
- 大量历史订单会影响查询速度

### 功能限制
- 暂不支持港股和美股
- 暂不支持实时数据
- 单次只能分析一只股票
